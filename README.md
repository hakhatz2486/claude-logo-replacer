# claude-logo-replacer

This userscript replaces Claude.ai's [default logo](https://lobehub.com/ja/icons/claude) with [Clawd](https://lobehub.com/ja/icons/claudecode), the character of Claude Code.

Claude.aiの[デフォルトのロゴ](https://lobehub.com/ja/icons/claude)を[Clawd](https://lobehub.com/ja/icons/claudecode)(Claude Codeのキャラクター)に置き換えるユーザースクリプトです。
