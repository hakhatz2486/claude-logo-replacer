# Claude Logo Replacer

English | [日本語](https://github.com/hakhatz2486/claude-logo-replacer/blob/main/README-ja.md)

A userscript that replaces the default Claude.ai logo and favicon with [Clawd](https://lobehub.com/ja/icons/claudecode), the character of Claude Code.

## Features

- Replaces Claude.ai's default logo with Clawd
- Replaces the browser favicon with Clawd
- Follows Claude.ai's dynamic page rendering
- Supports userscript managers such as Tampermonkey and Violentmonkey on Chromium-based browsers and Firefox

## Installation

1. Install a userscript manager such as [Tampermonkey](https://www.tampermonkey.net/) or [Violentmonkey](https://violentmonkey.github.io/).
2. Open [claude-logo-replacer.user.js](https://github.com/hakhatz2486/claude-logo-replacer/raw/refs/heads/main/claude-logo-replacer.user.js).
3. Confirm the installation in your userscript manager.
4. Reload `claude.ai`.

## Usage

After installation, the Claude.ai logo and browser favicon are automatically replaced with Clawd.

## Notes

- The script may stop working if Claude.ai changes its DOM structure or logo markup.
- This script operates only through the Claude.ai interface and does not use an API or API key.
