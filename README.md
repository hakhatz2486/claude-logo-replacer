# claude-logo-replacer

[README-ja](README-ja.md)

This userscript replaces Claude.ai's [default logo](https://lobehub.com/ja/icons/claude) and favicon with [Clawd](https://lobehub.com/ja/icons/claudecode), the character of Claude Code.

It is compatible with common userscript managers, including Tampermonkey and Violentmonkey on Chromium-based browsers and Firefox.
